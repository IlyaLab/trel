Transcriptional Regulation & Epigenetic Landscape Explorer

This is a tool for exploring and understanding the transcriptional regulation of cells.


The lefthand panels can be used to:


• Open a cell type vs cell type heatmap for a given gene.
• Open a gene vs cell type heatmap for a given cell type.

The top menu offers data annotations, the three dimensional epigenetic landscape, and additional information.


Heatmaps:


•Click on a row of a heatmap to generate the heatmap for the selected data feature (cell type or gene).
•A gallery of genes is available for comparison.
•Heatmap data can be exported as delimited text files.
